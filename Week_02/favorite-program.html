The Feedback Collector

This is one of the first programs I wrote. The program allows a user to enter multiple positive feedback phrases in one pass, separated by exclamation points, and will return numbered list of those phrases with common errors corrected. The program applies a capital letter to each word before putting into a numbered list. Finally, the program validates that a null value has not been entered and that at least one exclamation point has been entered. The user has a restart option if they wish to add more phrases.

Here is sample code which validates the user input before returning the numbered phrase list:


        def phrase_validation():
            valid_phrase = input('Enter as many phrases as you\'d like in a single line, then hit ENTER. '
                                 'You don\'t have to use capitalization. ')  # user enters input here
        while '!' not in valid_phrase or valid_phrase is None:  # validation loop. Blanks or missing '!' are not valid
            valid_phrase = input('I\'m not sure I understand. Please enter at least one phrase ending in a \'!\': ')
        valid_phrase_list = []  # empty list to capture phrases
        for phrase in valid_phrase.split('!'):  # split string at '!' (this removed the '!' - see processing for '!' add
            if phrase:
                valid_phrase_list.append(phrase.lower().strip().replace('  ', ' '))  # corrects simple user entry errors
        return valid_phrase_list
